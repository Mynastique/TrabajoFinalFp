# TrabajoFinalFp
"Aplicación web para gestionar un centro de estética, con funcionalidades CRUD, login de usuarios y gestión de citas"

## Tecnologías
-HTML5, CSS3, JavaScript
-GitHub
-Git Bash

## Instalación
1. Clonar el repositorio:
```bash
git clone https://github.com/Mynastique/TrabajoFinalFp.git
```
2. Abrir el archivo index.html en el navegador o usar la extensión Live Server de VS Code.
